# TriboLab-Analyzer
Программа обработки результатов трибологического эксперимента.
## Возможности

- загрузка CSV / TXT / XLSX;
- проверка корректности данных;
- предобработка (пропуски, выбросы, сглаживание, обрезка по времени);
- расчёт коэффициента трения μ и показателей изнашивания;
- статистика и доверительный интервал;
- графики и сравнение нескольких опытов;
- экспорт отчёта в Excel и PDF.

## Установка

```bash
py -m pip install -r requirements.txt
```

## Запуск

```bash
py main.py
```
